# Basic Network Configuration using Cisco Packet Tracer

## Project Overview

This project demonstrates the basic configuration of a small network using Cisco Packet Tracer.

## Objectives

- Configure IPv4 addresses for all network devices.
- Configure one Cisco router and two Cisco switches.
- Configure basic device security using passwords.
- Verify network connectivity.

## Network Topology

- 1 Router
- 2 Switches
- 2 PCs
- 2 Servers

## Tools Used

- Cisco Packet Tracer
- Cisco IOS

## Skills Practiced

- IPv4 Addressing
- Router Configuration
- Switch Configuration
- Device Security
- Network Connectivity Testing

## Result

Successfully configured the network and verified communication between devices.
